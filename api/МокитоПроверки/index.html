КоличествоВызовов​

Начинает утверждение для проверки количества вызовов метода.

Устанавливает параметры поиска вызовов.

Параметры метода

  • ИмяМетода
    • Строка - Имя метода проверяемого объекта
    • Произвольный - Вызов метода проверяемого объекта
  • ПараметрыВызова
    • Массив из Произвольный - Параметры вызова метода проверяемого объекта, необходим при указании имени метода

  • ИмяМетода

Равно​

Проверяет вызовы метода (см. КоличествоВызовов). Их количество должно быть равно переданному значение, иначе будет выброшено исключение.

Параметры метода

  • Количество
    • Число - Ожидаемое количество вызовов
  • Описание
    • Строка - Описание проверки, необходимо для идентификации упавшей проверки
